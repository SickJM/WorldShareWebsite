How to import a world into minecraft?

Unzip the world file

Download the world here then extract all the .zip file. Place this unzipped file somewhere that can be found easily.

Locate the saves file

Locate the .minecraft file on your device. Locate this by typing 'Windows + R' and type '%AppData%' then press .minecraft -> saves.

Import into Minecraft

Drag and drop the unzipped folder into the saves folder located inside of '.minecraft'. Load into minecraft than the world should be playable. If it is not please contact me here :)
